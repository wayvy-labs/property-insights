## Getting Started (Dev Setup)

### 1. Clone the Repo

```bash
git clone https://github.com/wayvy-labs/property-insights.git
cd property-insights
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Servers

```bash
npm start
```

- **Backend**: Node.js, Express.js, Postgresql
